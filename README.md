# Software_7A

## ★ コンパイルの流れ

コマンドライン上で"make all"を実行しサーバ側のプログラムとクライアント側のプログラムをコンパイル

できたクラスファイルを削除する場合は"make clean"

## ★ 実行の流れ

### サーバ側

java MultiClientServer と入力

### クライアント側

java TitleScreen と入力

↓

タイトル画面が表示される。

↓

Quitで終了

Keyで操作方法を表示

Loginで初期情報の入力へ

↓

テキストフィールドにユーザーネームを入力(スペースかアンダーバーのみの名前は名無し扱い)

↓

ボタンを押して外見を選択

↓

メイン画面が起動して

サーバーに接続する

### 操作方法

esc...メニューの表示/非表示

enter...チャットモードと移動モードの切り替え

方向キー...移動
